# OpenWeatherMap_app

Интерактивное веб-приложение для анализа температурных данных по городам на основе  
исторических данных (CSV или генерация) и текущей температуры через OpenWeatherMap API.

### Streamlit Cloud (деплой)
https://appopenweathermap-hg4xgveljxsinvdmrpguqs.streamlit.app/

### Локальный запуск
```bash
pip install -r requirements.txt
streamlit run app.py
```
## Файлы проекта

`app.py` — Streamlit интерфейс  
`utils.py` — логика анализа и API  
`tasks.ipynb` — аналитические задания  
`temperature_data.csv` — пример данных  
`requirements.txt` — зависимости  
`README.md` — описание проекта

### Сценарии работы приложения

| Сценарий                            | Что происходит                                                                                                 |
| ----------------------------------- | -------------------------------------------------------------------------------------------------------------- |
| CSV не загружен, API ключ не введён | Используются **сгенерированные исторические данные**. Текущей температуры нет.                                 |
| CSV загружен, API ключ не введён    | Используются данные из CSV для анализа истории. Текущей температуры нет.                                       |
| CSV не загружен, API ключ введён    | Используются **сгенерированные данные** + текущая температура берётся через API и проверяется на аномальность. |
| CSV загружен, API ключ введён       | Исторический анализ на основе CSV + текущая температура через API.                                             |
